# Configure Mapper (configure_mapper.py)

## Purpose
Generates a configuration file for the mapper, meaning the user doesn't have
to retype command line arguments every time they run the mapper.

## Functionality
Writes the following parameters to config/mapper.conf:

- Path to the binary being mapped
- Path to the text file containing the input parameter ranges
- Path to the text file containing the input parameters
- Path to the text file containing the default values for the input parameters
- Path to the reference output file (generated by the binary using default
input parameters)
- Path to the test output file (generated by the binary when it is run)
- Path to the directory containing the mapping orders (.json files)
- Arguments to pass to the binary when it is run
