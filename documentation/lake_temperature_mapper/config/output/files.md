# Files (APP/config/output/files.ini)

## Purpose
Configures the [file view.](../../src/output/views/file.md)

## Fields
All paths are relative to [APP_ROOT.](../../src/root.md)

| Field Name | Type | Description                                       |
|------------|:----:|---------------------------------------------------|
| samples    | Path | Path to the directory for outputing sample values |
| errors     | Path | Path to the directory for outputing model errors  |
| analysis   | Path | Path to the directory for analysis output         |
