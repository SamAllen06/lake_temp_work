# LakeTemperatureMod.F90
This is an updated LakeTemperature module that allowed me (Aidan) to compile elmtest for
the first time. The procedure was to open the docker image from Docker4SPEL, extract
[newlaketemperature.tar](../../2023/11/newlaketemperature.md), and replace
LakeTemperatureMod-NonCompact with LakeTemperatureMod in the Makefile.
